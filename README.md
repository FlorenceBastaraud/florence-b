# Mon site personnel

Développeuse Front-End, j'ai créé ce portfolio pour présenter mes projets et mon univers, où l'esthétique se marie à la fonctionnalité.

## Stack

Next.js 16 (App Router), TypeScript, Tailwind CSS et GSAP.

## Fonctionnalités

Animations légères, filtres projets, language switcher, curseur personnalisé, design responsive.

## Structure

- `app/` : pages et layout
- `components/` : les composants du site
- `context/` : contextes
- `hooks/` : hooks personnalisés
- `public/data/` : données JSON projets
- `public/images/` : images et miniatures

## Accès

Site en ligne : [florence-b.com](https://florence-b.com)  
Pour lancer localement :

```bash
npm install
npm run dev
```
